# DevOps СИТУ Лабораторная работа №3: Сканирование целевых хостов с помощью Ansible и NMAP

## Описание

Цель лабораторной работы — автоматизировать установку **NMAP** на удаленный хост с помощью **Ansible**, считать список целей из файла, выполнить сканирование каждой цели на открытый **TCP-порт 80** и вывести результаты сканирования на экран.

## Структура проекта

```text
.
├── inventory.ini
├── playbook.yml
├── targets.txt
└── README.md
```

## Содержимое файлов

### `inventory.ini`

Список целевых хостов.

```ini
[webservers]
172.16.10.3 ansible_user=runner

[target]
172.16.10.3
```

### `targets.txt`

Файл со списком целей для сканирования.

```text
moodle-epos.permkrai.ru
www.psu.ru
fast.com
```

### `playbook.yml`

Плейбук выполняет следующие действия:

1. Копирует файл `targets.txt` на удаленный хост.
2. Устанавливает пакет **NMAP**.
3. Считывает содержимое файла со списком целей.
4. Преобразует содержимое файла в список адресов.
5. Запускает команду `nmap -p 80` для каждой цели.
6. Собирает результаты сканирования.
7. Выводит результаты выполнения на экран.

## Запуск

Выполнить команду:

```bash
ansible-playbook -i inventory.ini playbook.yml
```

## Результат

После успешного выполнения плейбука:

- установлен пакет **NMAP**;
- файл со списком целей скопирован на удаленный хост;
- выполнено сканирование каждой цели на **TCP-порт 80**;
- результаты работы **NMAP** отображены в консоли для каждой цели.

## Используемые технологии

- Ansible
- NMAP
- Ubuntu/Debian
- APT
